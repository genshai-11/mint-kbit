# 01 — PRD

Status: **Accepted for clean React planning**

## Product intent
Build a credible multilingual KBIT public platform that preserves existing content and creates a future path to admin/content operations.

## Primary users
1. Prospective members and partners.
2. Event attendees and trainees.
3. KBIT staff/content operators, later phase.

## V1 must do
- Render EN/VI/KO public pages.
- Preserve transferred content/data fidelity.
- Keep gaps explicit: experts, testimonials, hero asset quality, production image hosting.
- Load quickly on public routes.
- Avoid carrying old Next.js implementation complexity.

## Out of scope for first transfer phase
- No Next.js reuse.
- No design-reference migration.
- No admin CRUD implementation.
- No production deploy.
- No deletion of existing target folders.
